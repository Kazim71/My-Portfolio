# Mohammad Kazim — Portfolio

Personal portfolio website showcasing my work as a Software Engineer.

## Tech Stack

- **Framework:** Next.js 15
- **Styling:** Tailwind CSS 4
- **Animations:** Framer Motion
- **UI Components:** Aceternity UI, Radix UI
- **Deployment:** Vercel
- **Analytics:** Vercel Analytics

## Getting Started

```bash
# Install dependencies
pnpm install

# Run development server
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) to view the portfolio.

## Deployment

This project is deployed on Vercel and auto-deploys on push to `main`.

**Live:** [https://my-portfolio-peach-delta-18.vercel.app](https://my-portfolio-peach-delta-18.vercel.app)
